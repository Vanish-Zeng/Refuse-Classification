# Refuse-Classification

## 项目名称

拉拉队立大功

## 主要功能

1. 通过对垃圾训练数据集进行深度学习，产生训练结果；
2. 将训练结果用于测试数据集进行测试，判断训练结果的准确率；
3. 同时在界面中加入自己导入图片的操作，并使用训练结果来分析图片是哪种类型的垃圾。

## 项目包含两部分内容

1. 基于DenseNet模型的深度学习训练垃圾数据集的算法
2. 垃圾分类系统的服务器端代码
